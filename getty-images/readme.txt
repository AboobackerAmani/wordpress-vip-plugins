== Description ==

Allow users to search, download, and insert images from Getty Images. Extends the WordPress Media Manager with a "Getty Images" mode that allows users to:

* Search for images
* Refine their searches by keywords or refinement categories
* Preview image positioning in their posts by inserting image comps
* Download images into WordPress media library (requires a Getty Images account)

== Installation ==

1. Upload getty-images to /wp-content/plugins/.
2. Activate plugin through the WordPress Plugins menu.
3. Click on "Getty Images" in edit post screen to browse the Getty Images library.
